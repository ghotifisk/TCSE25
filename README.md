# Introduction to the Tools of Scientific Computing

This repository contains the source code discussed in the 2nd edition of the  text [*Introduction to the Tools of Scientific Computing*](https://link.springer.com/book/10.1007/978-3-031-16972-4) by Einar Smith, published in 2022 as  part of the Springer-Verlag book series *Texts in Computational Science and Engineering* (TCSE, volume 25).
